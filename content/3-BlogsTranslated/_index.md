This section lists and introduces the technical blogs translated during the internship:

### [Blog 1](3.1-Blog1/)

**Unified Secrets Security with GitGuardian and AWS Secrets Manager**
This blog introduces a comprehensive secrets security solution by combining AWS Secrets Manager and GitGuardian through the ggscout utility. It analyzes the "visibility gap" in software development, the mechanics of anonymous cryptographic fingerprinting (HMSL), and a 5-step phased roadmap to automate detection, alerting, key rotation, and the continuous protection of Non-Human Identities (NHI) across Git repositories and AWS cloud environments.

### [Blog 2](3.2-Blog2/)

**AI-Powered Test Automation with Rapise and Amazon Bedrock**
This blog introduces the integration of Amazon Bedrock's Large Language Models (LLMs) into Inflectra's Rapise automated testing tool. It analyzes the bottlenecks of traditional test automation (UI changes, broken test scripts), explains the mechanics of UI contextual awareness and self-healing scripts, and provides a 5-step implementation roadmap to enhance QA productivity for enterprises.

### [Blog 3](3.3-Blog3/)

**AWS Transform – When AI Automatically Clears "Technical Debt" for Thousands of Repositories**
This blog introduces AWS Transform's new Continuous Modernization feature, which automates scanning, detecting, and remediating technical debt across thousands of repositories. It covers operational modes (Continuous and Campaign), pre-supported runtimes and SDKs, and integrations with AWS Security Agent and coding agents via the Model Context Protocol (MCP).
